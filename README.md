# object-detect
本仓库存放的是目标检测YOLO系列的一些代码以及改进模块的代码实现，需要的小伙伴自取就可以啦~

文件夹Attention Modules for improve YOLO中存放了不同的注意力机制
文件夹Conv Modules for improve YOLO中存放了不同的卷积模块
文件夹Different IoU for improve YOLO中存放了不同的IoU损失函数
